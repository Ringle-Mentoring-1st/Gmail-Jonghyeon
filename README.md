# Gmail Clone ✉️
**Apply Next.js, TypeScript, Tailwind**

## 1. Main functionalities
1. Compose first mail (inside of fake redux d)
2. Reply in thread
3. Classify mails by criteria, Starred / Inbox / Sent / Important
